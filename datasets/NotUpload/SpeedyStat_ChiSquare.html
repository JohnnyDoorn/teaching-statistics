We use the contigency table to assess whether there is an association between two categorical variables. When each of the two categorical variables has 2 levels, we can additionally look at the odds ratio, and even conduct a one-sided test. The sampling distribution of the X2 statistic is approximately normal from a certain cell size threshold (>= 5 observations in each cell). When this is not the case, we either use the continuity correction, of apply Fisher's exact test. Note that these latter two approaches can be applied in any scenario - there is no sample size minimum or maximum.

Independent Samples T-Test

jaspTTests::TTestIndependentSamples(
        version = "0.18.1",
        formula =  ~ numericPlacement,
        barPlot = TRUE,
        group = "Brand")

When we encode the "Placement" variable as 1's (correct) and 0's (incorrect), we can also analyze this data by means of a two-sample t-test. This test (which does have some t-test assumptions violated), then quantifies whether there is a difference in the proportion of correct placement, between the two brands. You can see below that for a two-sided test, the p-value is exactly the same as for the X2 statistic in the test above. With this framework, we could also do a one-sided test, to instead test the hypothesis that Maxicose has a higher proportion of correct placement. As with any one-sided test, if your alternative hypothesis guessed the correct direction of the effect, it will result in a lower p-value (i.e., more easiliy obtain significance). Of course, if the alternative hypothesis is wrong about the direction, you will probably not reject the null hypothesis. To understand this dynamic, download the jasp file and play around with the options for the alternative hypothesis (go from two-sided to each of the one-sided hypotheses).

Note: we can only do this for 2x2 setup, because that allows us to encode the two levels of one variable as 0 and 1. In turn this then enables us to make one-sided statements. When there are more than 2 levels to the factor variable, we can only speak about "differences". This is exactly the same dynamic as when we compare the ANOVA (>= 2 levels; F-test is two-sided) to the t-test (2 levels; t-test can be one-sided).
